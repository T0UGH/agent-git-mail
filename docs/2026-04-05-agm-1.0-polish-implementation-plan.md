# AGM 1.0 Polish Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 收口 AGM 1.0 的主入口、README、bootstrap、脚本与运行时文案，使项目从“已能跑的 MVP”升级为“入口统一、模型一致、文案可信”的工程收口型 1.0。

**Architecture:** 以 CLI `agm --profile <name> bootstrap` 作为唯一正式 onboarding 主入口，移除 shell wrapper 主路径；README 与 help 统一围绕 profile-first 心智重写；顺手清理 daemon/discovery/activation 文案中的 pre-profile 残留，并在低成本处补最小 identity guard。整个改造不新增能力，只统一入口与真相。

**Tech Stack:** TypeScript, Node.js, YAML config, git-backed mailbox repos, npm CLI, vitest
